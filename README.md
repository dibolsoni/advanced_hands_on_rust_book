# Advanced Hands-On Rust (Exercises & Experiments)

This repository contains my personal exercises, experiments, and
extended implementations based on the book *Hands-On Rust* by Herbert
Wolverson.

The goal of this project is to revisit the book using the latest stable
version of Rust and modern versions of Bevy, adapting examples where
necessary and expanding on the original concepts.

------------------------------------------------------------------------

## 📘 About the Book

*Hands-On Rust* focuses on:

-   Practical Rust programming
-   Game development fundamentals
-   ECS architecture
-   Procedural generation
-   Roguelike mechanics
-   Clean architectural patterns

This repository serves as a hands-on companion with updated
implementations.

------------------------------------------------------------------------

## 🚀 Objectives of This Repository

-   Rebuild book examples using the current Rust toolchain
-   Update rendering and ECS patterns to modern Bevy APIs
-   Refactor code to follow idiomatic Rust (2025+)
-   Explore architectural improvements
-   Experiment beyond the original book scope

This is not a verbatim reproduction of the book's source code. It is an
educational extension and modernization effort.

------------------------------------------------------------------------

## 🦀 Tech Stack

-   Rust (latest stable)
-   Bevy (latest compatible version)
-   Cargo workspace structure
-   Modern ECS patterns
-   Modular architecture

------------------------------------------------------------------------

## 📂 Repository Structure

Each chapter or concept is implemented in isolated modules or crates

## 🎯 Why This Exists

The Rust ecosystem evolves quickly.

This repository answers questions like:

-   How do the book's concepts translate to modern Rust?
-   How do we structure Bevy projects today?
-   What architectural improvements are possible?
-   How can we scale these examples toward production-quality systems?
